# Crypto Tools

A small collection of Bitcoin and crypto tooling focused on:

- Buy-Borrow-Die (BBD) strategy modelling
- Bitcoin-backed loan simulations
- Crypto reconciliation and tax experiments
- Long-term BTC collateral forecasting

## Live Demo

🚀 Launch the calculator here:

https://brightsgithub.github.io/Crypto_Tools/

## Included Tools

### Buy-Borrow-Die BTC Calculator

A Bitcoin collateral projection calculator inspired by platforms like Ledn.

Features include:

- BTC collateral growth projections
- Compound debt modelling
- LTV tracking
- Margin call and liquidation calculations
- Refinancing simulations
- Stress testing
- CSV export
- Tooltips explaining every calculation
