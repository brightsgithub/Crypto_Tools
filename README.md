# Crypto Tools

A small collection of Bitcoin and crypto tooling focused on:

- Buy-Borrow-Die (BBD) strategy modelling
- Bitcoin-backed loan simulations
- Crypto reconciliation and tax experiments
- Long-term BTC collateral forecasting
- Bitcoin macro floor & FX conversion tracking

## Included Tools

### Buy-Borrow-Die BTC Calculator
A Bitcoin collateral projection calculator inspired by platforms like Ledn. Features include BTC collateral growth projections, compound debt modelling, LTV tracking, margin call/liquidation calculations, refinancing simulations, stress testing, and CSV export.

### Bitcoin 200W SMA Blueprint
An analytical macro tracker mapping Bitcoin's structural support eras, total bear market durations, explicit weekly moving average touchpoints, and maximum capitulation deviations beneath the 200-week line floor.

### Bitcoin FX Calculator
An operational cross-currency utility built to compute precise purchasing power, execution entries, and currency conversion mechanics (USD/GBP pairs) when investing in Bitcoin.
